개인정보 처리방침

시행일: 2026-03-22 / 버전: 1.0

나를 축하해(이하 "앱")는 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수합니다.

1. 수집하는 정보

앱은 회원가입, 로그인, 연락처, 위치정보, 마이크, 카메라 권한을 요구하지 않으며, 사용자의 개인정보를 서버에 수집하거나 저장하 지 않습니다.

사용자가 앱 내에서 입력하는 문구, 선택하는 케이크 종류, 촛불 배치 등은 축하 화면 구성 기능을 위해 기기 내에서만 처리됩니 다.

2. 정보 이용 목적

입력된 정보는 다음 목적에만 사용됩니다.

  • 케이크 꾸미기 및 축하 화면 표시
  • 사용자가 입력한 메시지 반영
  • 앱 기능 제공 및 화면 구성

3. 제3자 제공 및 외부 전송

앱은 사용자의 개인정보를 제3자에게 제공하지 않으며, 외부 서버로 전송하지 않습니다.

4. 보관 기간

앱은 개인정보를 서버에 저장하지 않습니다. 사용자가 앱에서 입력한 내용은 앱 사용 중 화면 구성에만 활용되며, 별도의 회원 데이 터베이스로 관리되지 않습니다.

5. 아동의 개인정보

앱은 모든 연령이 사용할 수 있으나, 아동을 포함한 어떠한 사용자에 대해서도 개인정보를 별도로 수집하지 않습니다.

6. 개인정보 처리방침 변경

앱 기능 또는 개인정보 처리 방식이 변경되는 경우 본 처리방침을 업데이트할 수 있습니다.
