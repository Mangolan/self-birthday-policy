개인정보 처리방침

시행일: 2026-02-24 · 버전: 1.0

SELF BIRTHDAY(이하 “앱”)는 사용자의 개인정보를 소중하게 보호하며, 관련 법령을 준수합니다.

1. 수집하는 개인정보 항목

  • 사용자가 직접 입력하는 정보: 나이, 케이크 선택, 축하 메시지
  • 자동 수집 정보: 수집하지 않음

2. 개인정보 이용 목적

  • 맞춤형 축하 경험 제공
  • 앱 기능 운영 및 사용자 경험 개선

3. 개인정보 보관 기간

본 앱은 서버에 개인정보를 저장하지 않으며, 앱 종료 시 입력 정보는 즉시 삭제됩니다.

4. 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 보안 관련 조치

  • 사용자 입력 정보는 앱 기능 제공에만 사용됩니다.
  • 외부 서버로 전송하지 않습니다.
  • 로컬 저장이 필요한 경우 기기 내 저장소를 사용합니다.

6. 이용자의 권리

사용자는 앱을 삭제하거나 앱 내 저장 데이터 삭제 기능을 통해 개인정보를 삭제할 수 있습니다.
